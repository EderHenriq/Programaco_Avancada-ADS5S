# Projeto: Tela de Cadastro e Exibição (Trabalho Bimestral)

Este projeto foi desenvolvido como parte das atividades bimestrais da disciplina de Desenvolvimento Web, utilizando o framework **Angular 21**. O objetivo principal é demonstrar a manipulação de formulários, Data Binding e a renderização dinâmica de listas.

## 📋 Sobre o Trabalho
A aplicação consiste em uma Single Page Application (SPA) que permite:
1. **Captura de Dados:** Através de um formulário estruturado.
2. **Armazenamento em Memória:** Os dados são salvos em um array/lista no componente.
3. **Exibição Dinâmica:** Uma seção que lista todos os registros cadastrados em tempo real (utilizando a diretiva `*ngFor` ou `@for`).

## 🛠️ Tecnologias Utilizadas
* **Angular CLI:** v21.1.4
* **TypeScript:** Lógica de negócio e tipagem dos dados.
* **HTML5/CSS3:** Estrutura e estilização da interface.
